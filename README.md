# Work Day Scheduler

# About
A simple work day scheduler based on the average "9-5" job model. Schedule tasks throughout your day with a system that periodically changes the color of the tasks based on the current hour. It is able to take a user input and save it into the users local storage.

# Build
- Built uses the CSS framework "Bootstrap"
- jQuery allows for periodic changes to the HTML
- Moment.js is used to supply the date and time
